## 💻 Sobre o projeto

🚀 Alurapic é um projeto desenvolvido durante o curso de fundamentos de Angular da Alura.

[Alura](https://www.alura.com.br/curso-online-angular-fundamentos).

---

## 🏃‍♀️ Como rodar

- `cd alurapic > npm i > npm start`

- `cd api > npm i > npm start`

---
